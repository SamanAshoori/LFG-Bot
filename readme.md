# Discord GIF Bot

This bot sends a GIF when a message ends with `!!!`.

## Setup
1. Create a bot in the [Discord Developer Portal](https://discord.com/developers/applications).
2. Add the bot token as a secret in Replit:
   - Key: `DISCORD_TOKEN`
   - Value: Your bot token
3. Upload your GIF files to the project.
4. Run the bot!
